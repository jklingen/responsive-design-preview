Responsive Design Preview

This is a simple tool providing a quick preview of a website in various screen sizes. It supports pages running on your localhost, so it is a great companion for local development of reponsive websites.

Always be aware that this is just a preview tool. It can give you an idea about how different devices might render a web page and helps to recognize early when changing something for a specific devices breaks something on another device, but of course it is not the real thing. It is still important to do testing on different devices.
